# HTML Puzzle

Статический HTML-пазл, обернутый в Electron для запуска как desktop-приложение.

## Запуск в режиме разработки

```bash
npm install
npm start
```

## Сборка Windows .exe

```bash
npm install
npm run dist
```

Готовые файлы появятся в папке `dist`.

## Быстрая сборка без установщика

```bash
npm run pack
```

Этот вариант создаст распакованную папку приложения в `dist`.

## Сравнение с JigsawPlanet

Референс: https://www.jigsawplanet.com/?rc=play&pid=3ca3c7c750ce

### Что уже реализовано

- Загрузка своей картинки.
- Настройка количества частей через `X * Y`.
- Ограничение общего количества деталей до `5000`.
- Перетаскивание кусочков мышью.
- Слипание соседних частей.
- Перетаскивание соединенных групп вместе.
- Таймер сборки.
- Попап победы с итоговым временем.
- Кнопка подсказки-призрака.
- Кнопки `Упорядочить` и `Перемешать`.
- Electron-обвязка для запуска как desktop-приложение.

### Чего пока не хватает

- Прогресс сборки в процентах.
- Отдельный предпросмотр исходной картинки.
- Настройка прозрачности подсказки.
- Выбор позиции подсказки.
- Контур области картинки, если подсказка скрыта.
- Полноэкранный режим.
- Пауза таймера и игры.
- Горячие клавиши:
  - `A` - упорядочить;
  - `D` - перемешать;
  - `G` - показать или скрыть подсказку;
  - `I` - показать или скрыть предпросмотр картинки;
  - `P` - пауза.
- Выделение нескольких деталей рамкой.
- Добавление или удаление деталей из выделения через `Ctrl + click`.
- Режим click-to-pick: кликнуть деталь, вести без зажатой мыши, вторым кликом отпустить.
- Поворот деталей, если режим включен.
- Звуки соединения и победы.
- Настройка цвета фона.
- Автосохранение незавершенной игры.
- Список результатов или таблица рекордов.
- Отдельное меню настроек.
- Настройки формы деталей и режима игры.
- Более умная команда `Упорядочить`: раскладывать только свободные детали и не трогать уже соединенные группы.
- Более умная команда `Перемешать`: перемешивать только свободные детали и не трогать уже соединенные группы.
